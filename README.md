# Ejemplo implementación Api Rest
Ejemplo para la implementación de un Api Rest utilizando la siguientes tecnologias

* Node Js
* Mysql
* Sequelize

## Paquetes utilizados

* express
* body-parser
* helmet
* cors
* jsonwebtoken
* ajv
* mysql2
* sequelize
* sequelize-json-schema
